# OBS - Online Business Service

منصة لإضافة وعرض الإعلانات والخدمات على الإنترنت.

## الميزات
- تسجيل دخول بالبريد الإلكتروني
- إضافة إعلانات مع صور
- الإعلانات المميزة تظهر فوق الكل
- لوحة الأدمن للتحكم في الإعلانات والمستخدمين
- صفحة تواصل تحفظ الرسائل في Firebase
- تصميم حديث واحترافي

## روابط
- [رابط المشروع على GitHub Pages](https://اسمك.github.io/اسم-الريبوزيتوري/)

## كيفية الاستخدام
1. سجل دخولك أو أنشئ حساب جديد
2. أضف إعلانك مع الصورة والوصف
3. شاهد إعلاناتك في صفحة الإعلانات
4. الأدمن يمكنه تمييز الإعلان كمميز وحذفه
5. استخدم صفحة التواصل لإرسال رسائل
